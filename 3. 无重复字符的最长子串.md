# 3. 无重复字符的最长子串

###### 给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串** 的长度。



一开始没有思路，后来看了题解才知道。

这题需要关注到字符**子**串的**连续性**，利用这个性质借助队列的想法就比较容易求解。

```C++
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        string maxSubStr = "";
        int maxLength = 0;
        for(int i=0;i<s.length();i++)
        {
            char t = s[i];
            int index = maxSubStr.find(t);
            if(index == maxSubStr.npos)
                maxSubStr += t;
            else
            {
                int l = maxSubStr.length();
                if(l > maxLength)
                    maxLength = l;
                maxSubStr = maxSubStr.substr(index, l);//从index开始往后l个长度，多于字符串长度则截取到endl
            }
        }
        // cout<<maxSubStr;
        return maxLength;
    }
};
```
